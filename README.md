WorkESB
=======

WorkESB is an Entreprise Service Bus for industrial usage. 
The team leader is Marc Buils (http://www.marcbuils.fr), automation and natural interface engineer freelance for industrial services.

This is a fork of M²Bench, a project maked by a R&T team of MATIS Group (http://www.matis-group.com) and provided under Lesser GPL license in June 2012.
